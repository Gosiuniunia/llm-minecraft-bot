# LLM Minecraft Bot

## Overview
This repository contains the codebase for a Minecraft agent controlled by a Large Language Model (LLM). The system enables users to issue high-level commands in natural language (e.g. `/llm get diamond pickaxe`), which are interpreted by the LLM and translated into sequences of in-game actions.

The agent has access to the game state, including inventory and environment, allowing it to plan and execute tasks such as resource gathering, crafting, and exploration in an autonomous manner.

## Milestones

🔹 **Milestone 1 — Basic Bot Control**  
Implementation of a Minecraft agent capable of movement, resource collection, and crafting using predefined commands.

🔹 **Milestone 2 — LLM Integration**  
Integration of a local LLM responsible for interpreting user commands and generating structured action sequences.

🔹 **Milestone 3 — System Integration**  
Connection of the LLM with the Minecraft agent, enabling full execution of user-defined tasks within the game.
